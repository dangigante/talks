The Programming Editor

You are an editor that programs, not a programmer.*

* Most of the time

Things Your Editor Must Do

  • Syntax Highlighting
  • Syntax checking
  • Scripting
  • Code Completion
  • Plugin architecture
  • Multi-file editing

Syntax

Themeable
Your eyes are fickle
Smart highlighting
Highlight bad syntax, built-in functions, etc.
Syntax checking
Mark or provide list of compilation/parse errors

Scripting

Be able to extend your editor easily (Lua, Lisp, VimL, Python, etc.)

Be able to automate/macro q

Code Completion

  • Keyword completion ^N
  • Omnicompletion (intellisense) ^X ^O
  • Dictionary completion ^X ^K

Plugin Architecture

Your editor should support plugins. If it doesn't, wtf?

  • Easy to install
  • No dependency hell. I'm looking at you Eclipse.

Modality

What does it mean to be modal?

  • Multiple uses for damn-near every key on your keyboard based on context
  • You're editing more often than you're creating.
  • Glued to homerow. Get your damn hand off the mouse.
  • Fast, effecient movement and edits
  • Treat the file like a canvas, know where your brush is and where it needs to go

Registers - Copy/Paste

  • Copy - Yank text into a register. Like a copy.
    • ["<register>][N]y<motion> - Yank motion
    • ["<register>][N]Y - Yank lines
  • Paste - Paste text from a register
    • ["<register>][N]p - Paste after cursor
    • ["<register>][N]P - Paste before cursor

Example:


"ayw - ?
  2Y - ?
2"ap - ?

Tabs

I use tabs wrong, but I don't care because it works.

  • Create a new tab
    • :tabnew [file]
  • Move between tabs
    • [N]gt - Next tab, or Nth tab
    • gT - Previous tab

With a Quickness!

Movements

  • [N]w & [N]b - forward/backward N words
  • [N]} & [N]{ - forward/backward N "paragraphs"
  • % - Go to matching brace/bracket/paren
  • [N]f<char> & [N]F<char> - forward/backward to specified character

Editing Effeciently

Combine motions, commands and counts to be as effecient as possible

  • Abuse word objects
    • ci" - Change Inside Quote
    • cat - Change Around Tag
    • ci[ - Change Inside bracketset
  • Use the correct motion
    • 4dw VS df| VS D
